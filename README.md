# Task 2 - Automated Data Extraction (Web Scraping)

## Growfinix Data Science Internship

### Objective
This project demonstrates web scraping using Python. Data was extracted from a public website using BeautifulSoup and Requests, organized using Pandas, and exported to a CSV file.

### Tools & Libraries
- Python
- Requests
- BeautifulSoup
- Pandas
- Google Colab

### Website Used
https://books.toscrape.com/

### Data Extracted
- Book Title
- Price
- Rating
- Availability

### Files Included
- Task2_Web_Scraping_Growfinix.ipynb
- books_data.csv

### Outcome
Successfully scraped book information from the website and stored the extracted data in a CSV file for further analysis.
